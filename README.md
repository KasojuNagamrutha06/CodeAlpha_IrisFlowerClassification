# 🌸 Iris Flower Classification

## 📖 Overview

The Iris Flower Classification project is a Machine Learning application that predicts the species of an Iris flower based on its physical measurements. The model classifies flowers into one of three species:

- Iris Setosa
- Iris Versicolor
- Iris Virginica

This project demonstrates the fundamentals of supervised machine learning using the famous Iris dataset available in Scikit-learn.

---

## 🎯 Objectives

- Load and explore the Iris dataset.
- Train a machine learning classification model.
- Predict Iris flower species based on measurements.
- Evaluate model performance using test data.
- Understand basic classification concepts in machine learning.

---

## 🛠️ Technologies Used

- Python
- Scikit-learn
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 📊 Dataset Information

The Iris dataset contains:

- 150 flower samples
- 4 input features:
  - Sepal Length
  - Sepal Width
  - Petal Length
  - Petal Width
- 3 target classes:
  - Setosa
  - Versicolor
  - Virginica

---

## 🚀 Project Workflow

1. Import required libraries.
2. Load the Iris dataset.
3. Split data into training and testing sets.
4. Train the classification model.
5. Make predictions on test data.
6. Evaluate model performance.
7. Display results and accuracy.
